# Reward Chart

Visual reward system with stars and goals.

GTK4/Adwaita · Python · v0.1.5 · 26 translatable strings

Part of [Autismappar](https://autismappar.se) — free tools for children with autism, language disorders and ADHD.

## Install

### Ubuntu / Debian
```bash
sudo apt install beloningskartan
```
Repo: [yeager/debian-repo](https://github.com/yeager/debian-repo)

### Fedora
```bash
sudo dnf install beloningskartan
```
Repo: [yeager/rpm-repo](https://github.com/yeager/rpm-repo)

## Translation

Translated via [Transifex](https://app.transifex.com/danielnylander/beloningskartan/). 11 languages.
