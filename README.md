# Belöningskartan

Visuellt belöningssystem med stjärnor och mål.

GTK4/Adwaita · Python · v0.1.5 · 26 översättningsbara strängar

Del av [Autismappar](https://autismappar.se) — fria verktyg för barn med autism, språkstörning och ADHD.

## Installation

### Ubuntu / Debian
```bash
sudo apt install beloningskartan
```
Repo: [yeager/debian-repo](https://github.com/yeager/debian-repo)

### Fedora
```bash
sudo dnf install beloningskartan
```
Repo: [yeager/rpm-repo](https://github.com/yeager/rpm-repo)
