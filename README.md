# Belöningskartan

Visual reward system with stars and goals

Part of [Autismappar](https://www.autismappar.se) — free apps for autism, language disorders and ADHD.

## Install

### Ubuntu/Debian
```bash
sudo apt install beloningskartan
```

### Fedora
```bash
sudo dnf install beloningskartan
```
