# Belöningskartan

Visual reward system with stars and goals for children with autism and ADHD.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install beloningskartan
```

### Fedora/RPM
```bash
sudo dnf install beloningskartan
```
